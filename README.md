# 🚀 PROJECT OVERVIEW

🏏 This project is a **Professional Cricket Analytics Dashboard** developed using **Tableau**.

The dashboard provides powerful insights into bowling statistics across all cricket formats with an advanced modern UI inspired by sports broadcasting analytics dashboards.

✨ The project focuses on:
- Interactive Data Visualization
- Cricket Bowling Analytics
- Dashboard Designing
- KPI Analysis
- Professional UI/UX
- Sports Analytics Storytelling
